# Gett

Gett is a corporate ground transportation platform offering a REST API for booking on-demand and pre-scheduled rides, managing business travel accounts, handling employee records, accessing order receipts, and integrating with expense management systems.

**Developer Portal:** https://developer.gett.com/docs/introduction  
**Status Page:** https://gett2.statuspage.io/  
**Base URL:** https://business-api.gett.com  

## APIs.json

This repository contains an [APIs.json 0.19](https://apisjson.org/) profile for Gett covering:

- `apis.yml` — Primary APIs.json provider profile
- `plans/plans.yml` — API access plans and credential provisioning
- `rate-limits/rate-limits.yml` — Rate limit policies (16 req/s per account)
- `finops/finops.yml` — Financial operations, expense integration, and cost management
